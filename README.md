#### 用途介绍

在Hexo的Public目录下，生成站点校验所需的文件

#### 配置方法

``` yml
site_auth:
  baidu_path: 'xxxx.html'
  baidu_content: 'xxxx'
  weixin_path: 'xxxx.txt'
  weixin_content: 'xxxx'
  google_path: 'xxxx.html'
  google_content: 'xxxx'
  google_adsense_path: 'xxxx.txt'
  google_adsense_content: 'xxxx'
  sogou_path: 'xxxx.txt'
  sogou_content: 'xxxx'
```

#### 参数说明

上面的配置信息依次是：百度站长、微信公众号、谷歌站长、谷歌广告、搜狗站长的站点校验。
