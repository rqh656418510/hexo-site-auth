#### 用途介绍

根据 _config.yml 里的配置信息，生成站点校验所需的文件，支持百度站长、微信公众号、谷歌站长、谷歌广告、搜狗站长、必应站长、必应广告的站点校验。

#### 配置方法

``` yml
site_auth:
  baidu_path: 'xxxx'
  baidu_content: 'xxxx'
  weixin_path: 'xxxx'
  weixin_content: 'xxxx'
  google_path: 'xxxx'
  google_content: 'xxxx'
  google_adsense_path: 'xxxx'
  google_adsense_content: 'xxxx'
  sogou_path: 'xxxx'
  sogou_content: 'xxxx'
  bing_path: 'xxxx'
  bing_content: 'xxxx'
  bing_adsense_path: 'xxxx'
  bing_adsense_content: 'xxxx'
```

#### 参数介绍

- **xxx_path**： 文件路径
- **xxx_content**： 文件内容
